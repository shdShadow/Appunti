# Definizione
Una **grammatica** e' una quadrupla $G = \{V, T, P, S\}$ dove:
- V e' l'insieme delle variabili o categorie sintattiche
- T e' l'insieme dei [[Simboli Terminali]]
- P e' l'insieme delle [[regole-di-pruduzione| regole di produzione]]
- $S \in V$ e' lo start symbol
